# R2 Save Method

## Deployment

- Create an R2 bucket.
- Deploy [worker](./cloudflare/worker.js) to Cloudflare Workers.
- In `Variables and Secrets`, set `AUTH_TOKEN` as the access token.
- In `Bindings`, bind the R2 bucket to the Worker and set `Variable name` to `SUB_BUCKET`.

## Modify Config

- Set `save-method` to `r2`.
- Set `worker-url` to your Worker URL.
- Set `worker-token` to your Worker token.

## Get Subscriptions

- YAML subscription:

```text
https://your-worker-url/storage?filename=all.yaml&token=AUTH_TOKEN
```

- Base64 subscription:

```text
https://your-worker-url/storage?filename=base64.txt&token=AUTH_TOKEN
```

- Rule-based `mihomo.yaml` file:

```text
https://your-worker-url/storage?filename=mihomo.yaml&token=AUTH_TOKEN
```
